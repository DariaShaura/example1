При запуске необходимо указать 3 входных файла с данными: 
	A.csv (данные выгрузки из системы учёта времени), 
	B.csv (оценки по задачам), 
	C.csv (стоимость часа специалиста)

Пример запуска:
	python3 main.py A.csv B.csv C.csv
